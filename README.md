    - 1️⃣ What is the difference between var, let, and const?
    var- Can change and can be declared again.
    let- Value can change, but cannot be declared again in the same scope.
    const- Value cannot change after assigning.
    
    - 2️⃣ What is the spread operator (...)?
    the spread operator is used to expand or copy elements of an array or object.

    - 3️⃣ What is the difference between map(), filter(), and forEach()?
    map() - Used when you want to change every element.
    filter() - Used when you want to keep only elements that match a condition.
    forEach() - Loop through items and do something.

    - 4️⃣ What is an arrow function?
    An arrow function is a shorter way to write a function using the => symbol.

    - 5️⃣ What are template literals?
    Template literals are a way to create strings using backticks (` `) instead of quotes.
